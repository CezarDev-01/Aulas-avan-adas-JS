JavaScript

Estrutura de controle

As estruturas de controle em JavaScript são usadas para controlar o fluxo de execução do código.br
Elas permitem que seu programa tome decisões, repita instruções e execute ações condicionais.


Condicionais

As estruturas condicionais permitem que o código execute determinadas instruções com base em uma condição.


if, else if e else

Essa é a estrutura condicional básica. O bloco de código dentro de if será executado se a condição for verdadeira.
O else if adiciona mais condições, e o else é executado se nenhuma das condições anteriores for verdadeira.

switch

O switch é uma alternativa ao if...else if...else quando se tem muitas condições baseadas no valor de uma variável.
